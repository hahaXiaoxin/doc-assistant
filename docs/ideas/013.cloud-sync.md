# 云端同步(可选,E2EE)

- **ID**: ideas-013
- **状态**: proposed
- **优先级**: P3(版本主题:v0.9 / Phase 4)
- **来源**: `docs/ROADMAP.md` · §5
- **创建日期**: 2026-05-24

## 背景
跨设备同步记忆层数据。当前所有数据本地 IDB,换设备/重装即丢失。

## 提案
用户主动点按钮触发同步(不做自动后台同步)。
- 配置页"云同步"区块:用户填自定义 endpoint(自托管,不内置默认地址)。
- 协议:`POST /sync` JSON。
- **端到端加密**:本地密钥加密 `MemoryRecord` 后再上传;服务端只见密文。
- 冲突策略:`timestamp` LWW(Last-Writer-Wins)。

## 验收标准
- [ ] 配置页 endpoint 填写 + 主动同步按钮。
- [ ] 本地加密 → 上传 → 拉取 → 解密链路打通,密钥不离机。
- [ ] LWW 冲突路径单测覆盖。

## 备注
- **红线**:不得内置默认云端地址;同步协议中 apiKey 字段必须为空。
- 优先级最低,远期能力。
