# Skills · 可索引可调用的技能包

- **ID**: ideas-003
- **状态**: proposed
- **优先级**: P2
- **来源**: `docs/backlog.md` · B-004
- **创建日期**: 2026-05-24

## 背景
对标 Claude/ChatGPT skills、Cursor rules、Continue slash commands:技能 = 一段写给 agent 看的描述 + 可选 tool 组合 + 触发语义。需"按需检索"避免 system prompt 爆炸。本项目跑在浏览器,不能依赖外部 skill 中枢,索引必须本地化(可复用 `QwenEmbeddingProvider` + `DexieMemoryStore` 向量能力)。

## 提案
**MVP**:
- 定义 `Skill` schema(`id / name / description / triggerKeywords / triggerRegex / instructions / requiredTools / embedding`)。
- 两种来源:内置(`packages/skills/builtin/` 静态 JSON)+ 用户自定义(配置页"技能" Tab / JSON 导入)。
- 实现 `SkillIndex`:启动期向量化 description + keywords 入 IDB(新表 `skills` + `skill_embeddings`);规则召回 + 语义召回双通道取并集。
- 新增 `RelevantSkillsSource`(priority 待 ROADMAP 批新占位,不复用 40/50);命中 skill 注入其 `instructions` + 动态加载所需 tool。
- 斜杠命令 `/skill <name>` 强制激活;sidebar 显示"本轮命中技能"小徽章。

## 验收标准
- [ ] schema 校验严格(JSON Schema);skill 不得注入可执行 JS,`requiredTools` 必须已注册。
- [ ] 双通道召回 + topK=3 注入 system prompt;UI 徽章可点击查看 instructions。
- [ ] 内置 + 用户自定义两条路径打通;`/skill` 命令可用。
